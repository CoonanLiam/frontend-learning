Einführung in HTML

HTML (Hyper Text Markup Language) ist das Skelett jeder Webseite. Es gibt dem Inhalt eine logische Struktur.

Aufbau eines HTML-Dokuments

!DOCTYPE html definiert den Dokumenttyp als HTML5.

Der head-Bereich enthält unsichtbare Metadaten, während der body alles Sichtbare beherbergt.

Wichtige Tags

  • <strong> macht Text fett.
  • <em> macht Text kursiv.
  • <br> erzwingt einen Zeilenumbruch.
